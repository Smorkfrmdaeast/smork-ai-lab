🚀 SMORK AI LAB
SMORK AI LAB is a modern AI tools interface designed to simulate a real AI startup SaaS platform. The project showcases a futuristic user interface where users can interact with multiple AI-style tools such as a chatbot, text generator, prompt builder, and productivity utilities.
The platform focuses on clean frontend architecture, responsive design, and interactive UI components while maintaining a sleek dark theme with glowing neon accents inspired by modern AI products.
This project was built as part of my frontend development practice, demonstrating the ability to design and build professional web interfaces for AI-based platforms.
✨ Features
AI Text Generator interface
AI Chatbot UI with message bubbles
Prompt Generator tool
Code Helper interface
Content Idea Generator
Modern dashboard layout
Glassmorphism cards and glowing UI elements
Responsive design for mobile and desktop
Smooth hover effects and micro-interactions
🛠 Tech Stack
HTML5
CSS3
JavaScript
🎯 Purpose
The goal of this project is to explore modern UI/UX patterns used in AI platforms and build a professional interface similar to real AI SaaS products.
